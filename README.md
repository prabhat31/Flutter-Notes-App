# 📝 Flutter Notes App

A modern **Notes Application built with Flutter** that allows users to create, edit, delete, and search notes easily.
The app uses **SQLite (sqflite)** for local data storage and includes **Dark Mode support** for better user experience.

## ✨ Features

* ➕ Add Notes
* ✏️ Edit Notes
* 🗑 Delete Notes
* 🔍 Search Notes
* 🌙 Dark Mode
* 📱 Clean and Responsive UI

## 🛠 Tech Stack

* Flutter
* Dart
* SQLite (sqflite)
* Material UI

## 📚 Learning Purpose

This project was built as part of my **Flutter development learning journey** to understand:

* Local Database Integration
* State Management
* UI Design
* CRUD Operations
